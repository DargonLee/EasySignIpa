### EasySignIpa

重签名 iOS 的 IPA 文件的命令行工具 (IPA文件需要是咂壳后的包)

### 准备

- Mac 机器
- 安装 Xcode
- Python 3.7 以上环境

### 安装

- 本地安装

```python
git clone https://github.com/DargonLee/EasySignIpa
cd EasySignIpa
pip install .
```

- pip 安装

```python
pip install -U esign
```
